tags

Bismillah preprocessor uses an intern language, guess what? Also called Bismillah.

This tag system allows developers to write C/C++ code whenever they want.

Bismillah preprocessor allows you to use as many Bismillah blocks as you want.

Example
#include <iostream>
using namespace std;

int main() {
    // cpp code here...

    <bismillah>
        # bismillah code here...
    </bismillah>

    // cpp code here...
}
Translation
#include <iostream>
using namespace std;

int main() {
    // cpp code here...

    // a bismillah block started here...
        // bismillah code here...
    // a bismillah code ended here...

    // cpp code here...
}

variables

Based on TypeScript, Bismillah looks so similar when variable initialization is about.

Bismillah defines six basic data types (char, bool, int, float, number & double).

⚠️ If Bismillah does not know of any variable it will be translated anyways. This is because that variable could be initialized outside Bismillah.

⚠️ Bismillah preprocessor does not verify arrays indexes, so you will be notified that an index could be wrong.

Example
<bismillah>
    # variable initialization
    a:int = 1234
    b:int = 4321

    # value assignation
    a = b + a + c[i + j]
    b = a * 2
</bismillah>
Translation
// a bismillah block started here..
    // variable initialization
    int a = 1234;
    int b = 4321;

    // value assignation
    a = b + a + c[i+j];
    b = a * 2;
// a bismillah block ended here...
